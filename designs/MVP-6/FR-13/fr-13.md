# FR13 User ID & Action Log

## Description
This log is not intended to be viewed by users. It records information about the date and time of each successful user login, user logout, assignment creation, and submission. This information includes assignment ID, submission ID, and all user information. Unlike FR7, it does not include specific details about the assignments and submissions.
